Meme Generatorweb app

React, Javascript, CSS

GitHub Repo Live version

Ceci est un site web qui utilise CSS et React avec les Hooks useState et useEffect. Elle permet de générer une image aléatoire à chaque clique du bouton "Get a new meme image", Les deux inputs permettent de rajouter le texte que l'on veut en haut et en bas de l'image.
